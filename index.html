Development Notes

Cheat Sheet for React Native Developers

If you are using a physical Android device for your Development/Testing, use the below commands.

To enable Live/Hot Reloading:

adb shell input keyevent 82

To enable debugging via Chrome

adb reverse tcp:8081 tcp:8081

Cheat Sheet for GIT Development

To preview Markdown (MD) file

VS Code Shortcut CTRL + SHIFT + V

Cheat Sheet for ATOM Development

To run Java programs:

After installing the SCRIPT Theme, use CTRL+SHIFT+B to run the code

Cheat Sheet for MySQL Development

To execute all SQL files in a folder:

cat *.sql | mysql -u root -p

To execute single SQL file in a folder:

cat YourFileName.sql | mysql -u root -p
